# Auth Microservice
> Microservice Backend managing `todos` collection

## Working without Docker
### Install
```sh
$ git clone
$ nvm use 10
$ npm install
```

### Get started
```sh
$ npm run dev
```

### Run Test
```sh
$ npm test
```
### Build Prod
```sh
$ npm run build
```
